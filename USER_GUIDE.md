# User Guide - نظام إدارة الاستلام المختبري

## Getting Started

### Login

1. Open the application
2. Enter your username and password
3. Click "Login"

### Dashboard

The dashboard shows:
- Receipt statistics (total, today, this week, this month)
- Pending and completed receipts
- Organization and user counts
- Recent activity
- Backup status
- Sync health

## Receipts

### Creating a Receipt

1. Click "New Receipt" or navigate to Receipts
2. Fill in receipt details:
   - Transaction type
   - Sender/Receiver organizations
   - Sender/Receiver names and job titles
   - Authorization number and date
   - Transaction date
   - Notes
3. Add items with sample types and counts
4. Click "Save"

### Updating a Receipt

1. Open the receipt
2. Click "Edit"
3. Modify fields as needed
4. Add/remove items
5. Click "Save"

### Receipt Statuses

| Status | Description |
|--------|-------------|
| Draft | Initial status |
| Approved | Verified and approved |
| Rejected | Not accepted |
| Archived | Stored for reference |
| Cancelled | Voided |

### Searching Receipts

1. Use the search bar on the Receipts page
2. Search by receipt number
3. Filter by status

## Organizations

### Viewing Organizations

1. Navigate to Organizations page
2. View list of all organizations

### Creating an Organization

1. Click "Create Organization"
2. Enter name, code, address, phone, email
3. Click "Save"

## Users

### Viewing Users (Admin/Supervisor)

1. Navigate to Users page
2. View list of all users

### Creating a User (Admin)

1. Click "Create User"
2. Enter username, full name, password, role
3. Click "Save"

### Changing Password

1. Click "Change Password" in the header
2. Enter current password
3. Enter new password
4. Confirm new password
5. Click "Change"

## Reports

### Viewing Reports

1. Navigate to Reports page
2. Select report type:
   - Summary report
   - Daily report
   - Monthly report
   - Institution statistics
   - Rejection statistics

### Exporting Reports

1. Select report parameters
2. Click "Export"
3. Choose format: CSV, XLSX, or PDF

## Backup

### Creating a Backup

1. Navigate to Backup page
2. Click "Create Backup"
3. Wait for completion

### Restoring a Backup

1. Navigate to Backup page
2. Click "Restore" on desired backup
3. Confirm restoration

### Verifying Backups

1. Navigate to Backup page
2. Click "Verify All"
3. View verification results

## Audit Log

### Viewing Audit Logs

1. Navigate to Audit page
2. View list of all system actions
3. Filter by action type

## Settings

### Changing Settings (Admin)

1. Navigate to Settings page
2. Modify desired settings
3. Click "Save Settings"

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| F5 | Refresh current page |
| Ctrl+N | New receipt |
| Ctrl+S | Save current form |
| Esc | Close dialog |

## Troubleshooting

### Cannot Login

1. Check username and password
2. Verify account is active
3. Contact administrator

### Receipt Not Saving

1. Verify all required fields are filled
2. Check item counts match totals
3. Verify you have permission

### Slow Performance

1. Check database connection
2. Verify system resources
3. Contact administrator
